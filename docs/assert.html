Assert

global class Assert
Description A set of assertion methods for writing Cinnamon tests. You should use Cinnamon's assert methods to assert a condition in your Cinnamon tests rather than using Apex's standard System.assert methods.

Methods

assert

global static void assert ( String message, Boolean condition)
Description Asserts that the specified condition is true. If it is not, the method returns an error that causes Cinnamon test execution to halt. The tearDown() method in your Cinnamon test still gets executed even when the assertion failure halts test execution.
Parameters
message Custom message displayed as part of the error message.
condition The condition to be evaluated.
