ElementLocator

global virtual with sharing class ElementLocator
Description ElementLocator specifies which element a WebElement object refers to.
A valid CSS selector must be provided to the ElementLocator constructor when instantiating an ElementLocator object.

For example, if you want to instantiate a WebElement object that refers to an INPUT field with class "foo" for a custom HTML page, you can use ElementLocator as follows: 


public with sharing class CustomObjectPageObject extends PageObject {
    WebElement inputField;
    public void initializePageObject() {
        inputField = getElement(new ElementLocator('input.foo'));
    }
}
