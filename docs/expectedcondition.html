ExpectedCondition

global interface ExpectedCondition
Description ExpectedCondition models a condition that evaluates to true or false at some point during test execution.

For example, you can use ExpectedCondition to test if a web page has loaded or if a certain element on the page becomes visible or invisible.
See Wait

Methods

   apply

String apply()
Description The returned String is a JavaScript snippet that is then executed on the Selenium server. The JavaScript tests for the expected condition and is formatted as an anonymous, self-executing function that explicitly returns a boolean. For example, to test if an element is in the DOM, use: 

 (function(){
   var el = document.querySelector('#id');
   return (el != null);
 }())
Returns A JavaScript snippet to execute on the Selenium server.
