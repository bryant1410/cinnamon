OrgUnderTest

global with sharing class OrgUnderTest
Description OrgUnderTest is a class that provides a set of utility methods to execute operations against an organization under test. All operations from this class are executed via Salesforce Tooling or the REST API.

This class must be used when an organization that tests are running against is different from the one where Cinnamon is installed.
