OrgUnderTest

global with sharing class OrgUnderTest
Description OrgUnderTest is a class that provides a set of utility methods to execute operations against an organization under test. All operations from this class are executed via the Salesforce Tooling API or REST API.

This class must be used when the organization under test is different from the one where Cinnamon is installed.
