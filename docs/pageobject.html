PageObject

global virtual with sharing class PageObject
Description PageObject is an abstract base class that is used to define a specific PageObject class for a UI page. All PageObject classes in Cinnamon must inherit from this class.

To instantiate a PageObject class, pass the type of PageObject class in to the Context.getPageObject() method as an argument. 


   public class TestCustomPage extends BaseTest {
       private CustomPageObject customPage;
       public override void setUp(Context context) {
           customPage = (CustomPageObject) context.getPageObject(CustomPageObject.class);
           customPage.initializePageObject();
       }
       public override void test(Context context) {}
   }

global virtual String getPathAndQuery(Map<String, String> params)
DescriptionReturn the initial URL path and query part for the Page Object. This method is often used to indicate the first URL the browser loads when implementing BaseTest.getStartingPath(). For example: 

   public with sharing class TeamDetailPageObject extends cinnamon.PageObject{
      public override String getPathAndQuery(Map params) {
      PageReference page = new ApexPages.StandardController(params.get('team')).edit();
      return page.getUrl();
      }
   }
