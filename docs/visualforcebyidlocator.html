VisualforceByIdLocator

global with sharing class VisualforceByIdLocator extends ElementLocator
DescriptionVisualforceByIdLocator is used to locate an HTML element generated by a Visualforce tag. This locator should be used when you want to locate an element that is not supported by VisualforceLocator.

The following code snippet shows how to retrieve a form WebElement generated by <apex:form> tag with 'storyWorkForm' set to Id attribute. Because VisualforceLocator currently does not support <apex:form> tag, VisualforceByIdLocator is used instead and 'form' HTML tag is passed in as the first argument. 

 
 cinnamon.WebElement storyWorkForm = getElement(new cinnamon.VisualforceByIdLocator('form', 'storyWorkForm'));
